# Task-Tracker
"This task tracker application, inspired by Jira's robust project management capabilities, empowers teams to efficiently manage tasks and projects."
"Built using HTML, CSS, and React, this application provides a user-friendly interface for creating, assigning, and tracking tasks across teams."
"Key features include task creation, assignment to team members, status tracking, and intuitive filtering for enhanced productivity."
"Designed with scalability in mind, it offers a seamless experience for small to medium-sized teams looking to streamline their project workflows."
"Explore the power of modern web technologies combined with agile project management principles, tailored for developers and project managers alike."
 
 
- **Auth Module**:
 
  - User login using email-based authentication.
  - Verification using Password.
 
- **User Module**:
 
  - The user module, implemented using React.js, facilitates seamless functionalities for adding, retrieving, and updating user information within the application, ensuring efficient management of user data.
 
- **Project Module**:

  - The project module utilizes React.js to facilitate seamless project management, featuring functionalities for creating, fetching, and updating project details within the application..
  
- **Task Module**:
  
  - In this task tracker application, administrators have the ability to create tasks assigned to multiple users within specific projects, streamlining project management and fostering collaboration across teams.
  - Users can view assigned tasks within their projects and update task statuses upon completion, ensuring transparency and progress tracking throughout the project lifecycle.
 
## Features

- User login and authentication
- User management with role-based access control
- Project management for organizing tasks
- Task creation and assignment across projects
- Task status tracking and updates
 
## Prerequisites
 
- Node.js
- npm/bun
  
 
## Installation
 
1. Clone the repository:
 
   ```bash
   git@github.com:Tamilselvam-Muthusamy/Task-Tracker.git
   ```
 
2. Navigate to the project directory:
 
   ```bash
   cd Task-Tracker
   ```
 
3. Install dependencies:
 
   ```bash
   npm install
   ```
 
   or
 
   ```bash
   bun install
   ```
   ```
 
## Usage
 
1. Start Command:
 
   ```bash
  npm run dev
   ```
 
   or
 
   ```bash
   bun run dev
   ```
